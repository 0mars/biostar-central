What is a recipe?
A "bioinformatics recipe" is a collection of computer code that performs a bioinformatics data analysis task.
What types of recipe will I find here?
We cover a wide variety of subjects, from RNA-Seq analysis, variant calling, classifications and many others. Some recipes reproduce published results , others demonstrate typicall use cases. A large number of recipes have educational purpose.
Where can I find recipes?
Visit the Project List page. Each project is a collection of recipes and results.You may also want to use the search box on top.
Where can I see the results of a recipe?
Yes, go the the "results" tab and see all the files the recipe has created.
Can I run a recipe on my own computer?
Yes! Recipes are generic and universal. The use of recipes is not restricted to this site. In the "Code View" select download script.
Is there a beginner friendly guide to recipes?
The Biostar Handbook has detailed explanation for many recipes. It is also the a resource that can train you in the use of bioinformatics tools.
What is the main purpose for recipes?
Recipes were designed address the challenges of reproducibility. It allows scientists to document, execute and share data analysis scripts and the results of running these scripts.
Can I make my own recipe?
You may adapt, modify and remix recipes to match your needs. Then, if want to you may share these on the same site and show others how you solved a problem. By supporting these interactions, we aim to foster a collaborative environment that facilitates creativity, efficiency and reproducibility.
What kind of code may a recipe contain?
A recipe may use any code that can be executed at command line, for example bash, python, R, makefiles and many others.
What software runs this site?
The software that runs this site is called theBiostar Engine and it was designed to generate an easy-to-use interface to command line scripts. In addition the site collects the files that are generated during the run of a script and shows them to the user in an visual manner.
