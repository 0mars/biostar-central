What is a recipe?
A "bioinformatics recipe" is a collection of commands used to perform a bioinformatics data analysis task. A recipe can be thought of as a "bioinformatics workflow" or a "bioinformatics pipeline".
How are recipes used?
Some recipes reproduce published results, others demonstrate typical use cases. Several recipes have been developed to support bioinformatics education.
How are recipes organized?
Visit the Projects page. Each project is a collection of recipes and results. You may also use the search box on top.
Can I run a recipe on my computer?
Yes! All recipes can be run at the command line on your system.
