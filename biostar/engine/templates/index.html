What is a recipe?
A bioinformatics recipe is a collection of commands used to perform a bioinformatics data analysis task.
What types of recipes are distributed here?
We cover a wide variety of subjects, from RNA-Seq analysis, variant calling, classifications and many others. For example:
Bacterial Variant Calling pipeline Alignment based RNA-Seq analysis Antisense transcript detection
Some recipes reproduce published results, others demonstrate typical use cases. A large number of recipes have been developed to support bioinformatics education.
Where are the recipes?
Visit the Project List page. Each project is a collection of recipes and results. You may also use the search box on top.
Where can I see the results of a recipe?
Go the the "Recipe Results" tab to see an example run for the recipe. All the outputs and results that a recipe creates will be visible there.
Can I run a recipe on my own computer?
Yes! Recipes are generic and universal. The use of recipes is not restricted to this site. In the "Code View" select download script.
Is there a beginner friendly guide to recipes?
The Biostar Handbook has detailed explanation for many recipes. The book is also the a resource that can train you in the use of bioinformatics tools.
What is the main purpose for recipes?
Recipes were designed address the challenges of reproducibility. Receipes allow scientists to document, execute and share data analysis scripts and the results of running these scripts.
Can I make my own recipe?
You may adapt, modify and remix recipes to match your needs. Then, if want to, you may share your approach on this same site and show others how you solved a problem.
What kind of code may a recipe contain?
A recipe may use any type of code that can be executed at command line. For example bash, python, R, Makefile and many others are all supported.
What software runs this site?
The software that runs this site is called theBiostar Engine and it was designed to generate an easy-to-use interface to command line scripts.
