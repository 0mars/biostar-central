Bioinformatics Recipes: Reproducible Data Analysis
==================================================

Bioinformatics Recipes is a `Python <http://www.python.org/>`_ and `Django <http://www.djangoproject.com/>`_ based
data analysis platform.

It is a simple, generic, flexible and extensible software that connects computational experts with end users.
